# Agrotools Forms

### API RESTful Node.js

Front-End:<br />
https://github.com/gabriel-cardoso-oliveira/agrotools-forms-client<br />

## Quick start

Install:

```
git clone https://github.com/gabriel-cardoso-oliveira/agrotools-forms-server.git
```
```
cd agrotools-forms-server/
```
```
yarn install
```
```
yarn knex:migrate
```

Test:

```
yarn run dev
```
